Join
==========VB==========
Function Join(SourceArray, [Delimiter]) As String

VBA.Strings 的成员

Join array elements into a string
==========百度百科===========
语法
Join(list[，delimiter])

Join 函数的语法有以下参数：
参数		描述
list		必选。包含要联接的子字符串一维数组。
delimiter	可选。在返回字符串中用于分隔子字符串的字符。如果省略，将使用空字符 ("")。 如果 delimiter 是零长度字符串，则在同一列表中列出全部项，没有分界符。
